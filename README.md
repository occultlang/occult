> [!NOTE]
> Minor-versions will be released as stable builds as new features are added and completed! <br><br>
> *All code may break with each release, as this langauge is **not** production ready at all yet.*

**<ins>Alpha 2.1.1 Release</ins>**
- Structs
  
**<ins>Alpha 3.0.0 Release</ins> (Mid 2026 - Early 2027)**

- Fix Linker 
- Borrow Checker
- New Intermediate Representation (SSA) for optimization pass
